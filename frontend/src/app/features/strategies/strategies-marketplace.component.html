How to create a strategy
Strategies are exported pipeline snapshots, not editable documents.
  1. Build and verify the pipeline you want to share.
  2. Export it as a strategy from the Pipelines workspace.
  3. Keep it private or submit the saved snapshot for marketplace review.
account_tree
Snapshot fidelity
Using a strategy recreates the exported pipeline snapshot instead of translating from prose.

Marketplace viewers see a rendered strategy presentation, but the saved source of truth is still the pipeline blob. That keeps exported and recreated pipelines aligned.
