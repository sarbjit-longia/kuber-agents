# Harmonic Patterns

Harmonic patterns are advanced chart patterns that use Fibonacci ratios to identify precise reversal points. They all follow a similar XABCD structure but differ in the specific Fibonacci ratios at each leg.

## Quick Reference

| Pattern | AD Retracement of XA | D vs X | Key Feature |
|---------|---------------------|--------|-------------|
| [Gartley](gartley.md) | 78.6% | Inside X | Most common harmonic |
| [Bat](bat.md) | 88.6% | Inside X | Deeper retracement |
| [Butterfly](butterfly.md) | 127.2-161.8% | Beyond X | D extends past X |
| [Crab](crab.md) | 161.8% | Far beyond X | Most extreme pattern |
| [AB=CD](ab-cd.md) | — | — | Simplest harmonic (symmetry) |
| [Three-Drive](three-drive.md) | — | — | Three pushes at Fib extensions |
